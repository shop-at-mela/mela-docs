# Mela Marketplace: Category Structure Deep Analysis

## Executive Summary

This document provides a comprehensive analysis of marketplace category structures for Mela, an Indian brands marketplace targeting international consumers in the US, UK, Canada, Australia, and the Middle East. Based on research of global platforms (Amazon, eBay), Indian-focused marketplaces (Jaypore, Fabindia), and export/demand data, we recommend a category structure optimized for discoverability while maintaining simplicity.

**Constraint**: Each L0 category has ≤30 total L2 subcategories across all L1s.

**Design Principles:**
- Each L0 has ≤30 L2 subcategories (max is 26)
- Categories = product types (nouns); Attributes = characteristics (adjectives/techniques/materials)
- US-friendly terminology with Indian terms in parentheses for discoverability
- Focus on India's export strengths; removed commodity/non-differentiated categories
- High-value, shelf-stable products that justify international shipping costs
- Cross-category filtering via attributes (Occasion, Technique, Material, Certification)
